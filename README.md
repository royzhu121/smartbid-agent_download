# SmartBid 智慧招标代理客户端

SmartBid 是面向工程招标代理业务的 AI 工作台，帮助招标代理团队把项目资料整理、招标文件审查、清标核查、资审核对、费用测算等日常工作集中到一个桌面客户端里完成。

桌面客户端适合在本地电脑上处理项目资料，减少重复录入和人工核对压力，让项目经理、招标专员和复核人员可以围绕同一套项目数据协同工作。

## 适用场景

- 招标文件审查与问题梳理
- 投标文件清标、资审资料核查
- 项目资料归档与过程留痕
- 工程咨询、招标代理相关费用测算
- 团队内部复核与结果导出
